== Description ==

Allows you to create a div or anchor or Date by using the shortcodes, Date [showdate] and end [end-showdate], start [a] and end [end-a] and start [div] and end [end-div]. To add an id of "foo" and class of "bar", use [div id="foo" class="bar"] and href in anchor like link="any link to redirect", use [a id="foo" class="bar" link="#"].


Up coming version will have many more options

Found a bug or have a feature request ? Report here or have any question contact us at <a href="mailto:info@sksdev.com">info@sksdev.com</a>


== Installation ==

1. Put the "Sksdev All Shortcode.zip" extract to plugin folder "/wp-content/plugins/" or upload it.
2. Go into the WordPress admin pannel and activate the plugin
3. Enjoy
